澈析 BI 快速开始

从创建账号到开始分析，了解产品的基本使用路径。具体可用功能取决于账号角色和管理员授权。

注册与登录

  1. 访问澈析 BI 注册页面并创建账号。
  2. 完成账号验证后登录应用。
  3. 进入首页后，查看管理员为当前角色开放的功能。

访问数据

数据源和数据集通常由管理员统一配置。普通用户只能访问被授权的数据内容。如果看不到预期数据，请先确认账号所属角色和授权范围。

创建图表

  1. 选择有权访问的数据集。
  2. 根据分析目标配置指标、维度、筛选条件和时间范围。
  3. 选择适合的数据可视化类型并预览结果。
  4. 保存图表，并根据需要加入仪表盘。

使用仪表盘

仪表盘可以组合多个图表，并通过筛选器帮助用户从不同角度观察业务数据。你看到的内容始终受账号权限和数据访问规则约束。

使用 AI Agent

在 Agent 入口中使用自然语言描述分析问题。建议说明业务目标、时间范围和关注指标，并对重要结论进行人工确认。

Agent 的回答用于辅助分析，不应替代业务判断、财务审计或其他专业决策。

权限说明

当页面、菜单、数据集或仪表盘不可见时，通常表示当前角色未获得相应权限。请联系空间管理员核实授权，不要通过共享账号规避权限控制。
