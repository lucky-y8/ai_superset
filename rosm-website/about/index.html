ABOUT CHEXI BI

让更多人清楚地理解和使用数据

澈析 BI 是一款融合 AI Agent、数据可视化与权限隔离能力的智能 BI 工具。

我们在做什么

数据分析工具功能日益丰富，但业务问题、数据口径和权限边界仍然容易分散在不同环节。澈析 BI 希望把自然语言提问、可视化探索和团队权限放在同一个分析环境中。

产品基础

澈析 BI 基于 Apache Superset 深度开发，在成熟的开源数据可视化能力之上增加权限隔离与独立 Agent 服务。Apache Superset 是 Apache Software Foundation 的开源项目；澈析 BI 是独立产品，并非 Apache Software Foundation 官方发行版本。

产品原则

我们重视清晰的数据访问边界、可验证的分析结果和真实可用的工作流程。AI 是分析助手，不替代组织的数据治理，也不替代用户对关键业务结论的判断。
