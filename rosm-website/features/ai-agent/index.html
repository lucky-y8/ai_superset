从问题开始，而不是从配置开始

面对临时分析需求，用户可以先用自然语言描述目标、范围和关注点。Agent 将对话作为分析入口，降低理解复杂工具界面的成本。

Agent 服务与澈析 BI 的数据分析环境协同工作。分析过程围绕用户可以访问的数据展开，减少在多个系统之间切换和重复解释上下文。

AI 能力不应绕过数据访问规则。澈析 BI 将权限隔离纳入产品设计，使 Agent 与图表、仪表盘处于一致的数据访问边界中。
